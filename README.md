# HR Analytics – Employee Attrition Analysis

## Overview
Analyzed ~1,470 employee records to identify key drivers of attrition and provide actionable insights to improve employee retention. This project replicates a real-world HR analytics use case focused on reducing employee attrition and enabling data-driven decision-making.

## Problem Statement
Employee attrition increases hiring costs, reduces productivity, and leads to loss of organizational knowledge. This project identifies key factors driving employee turnover and highlights high-risk employee segments.

## Tools Used
- Python (Pandas)
- SQL (PostgreSQL)
- Power BI

## Analysis
- Cleaned and preprocessed employee data using Python  
- Created derived features such as income groups and attrition flags  
- Performed exploratory data analysis to identify patterns and trends  
- Used SQL queries to analyze attrition metrics across multiple dimensions  

## Key Insights
- Employees working overtime show ~30–35% attrition vs ~10% for non-overtime employees, making it the primary driver of attrition  
- Low-income employees show ~25% attrition, nearly 2x higher than high-income groups (~10–12%)  
- Sales department shows the highest attrition (~20%), followed by HR (~18%) and R&D (~15%)  
- Poor work-life balance (levels 1–2) shows ~30%+ attrition, significantly higher than better balance levels  

## Business Impact
Insights from this analysis help HR teams:
- Reduce employee attrition and hiring costs  
- Improve workforce planning and retention strategies  
- Identify high-risk employee segments proactively  
- Enhance overall employee satisfaction  

## Recommendations
- Implement workload balancing strategies to reduce overtime dependency  
- Re-evaluate compensation structure for low-income employees  
- Focus retention efforts on high-attrition departments and roles  
- Improve work-life balance through flexible policies  

## Dashboard
The Power BI dashboard highlights key attrition drivers such as overtime, salary, department, and work-life balance, enabling quick identification of high-risk employee segments.
